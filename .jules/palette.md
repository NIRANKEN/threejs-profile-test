
## 2023-10-27 - [3D Accessibility] **Learning:** 3D objects in Canvas lack native DOM semantics, making them invisible to screen readers and keyboard navigation. **Action:** Added visually hidden `<button>` elements embedded via `@react-three/drei`'s `<Html>` component. By applying an `.sr-only` class to these buttons and mapping their `onClick` events to the 3D interaction logic, screen readers can announce interactive elements like "ベッドの詳細を表示" and keyboard users can navigate to them using 'Tab'. This ensures that the 3D space is accessible without breaking visual immersion.
