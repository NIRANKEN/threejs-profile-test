# UI/UX & Portfolio Reviewer Context

## Expertise

- Responsive design for creative portfolios.

- Tailwind CSS optimization and modern UI patterns.

- Framer Motion for HTML-based animations.

- Web Accessibility (A11y) in canvas-heavy apps.

## Review Guidelines

- Portfolio must be mobile-friendly. Ensure touch targets are at least 44x44px.

- Maintain a consistent aesthetic between 3D elements and HTML overlays.

- Check font legibility against various background colors (especially the dark/light mode toggle).

- Ensure smooth transitions when switching between "Room view" and "Panel view".
